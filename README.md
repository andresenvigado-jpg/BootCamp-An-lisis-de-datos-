## 📌 Descripción

Este repositorio es un espacio de aprendizaje y práctica desarrollado en el marco del **Bootcamp de Análisis de Datos de Talento Tech**. Aquí se documentan ejercicios, notebooks y proyectos construidos a lo largo del programa, con el objetivo de consolidar habilidades en el análisis, manipulación y visualización de datos.

> 🚀 *Repositorio en construcción — el contenido se irá actualizando a medida que avance el bootcamp.*

---

## 🎯 Objetivos del Repositorio

- Documentar el progreso y aprendizaje durante el bootcamp.
- Practicar conceptos clave de análisis de datos con Python.
- Organizar notebooks de Google Colab de forma estructurada y reutilizable.
- Construir un portafolio de proyectos educativos.

## 🚀 ¿Cómo usar este repositorio?

### Opción 1 — Google Colab (Recomendado)

1. Abre cualquier archivo `.ipynb` en este repositorio.
2. Haz clic en el botón **"Open in Colab"** (si está disponible) o copia la URL del notebook.
3. En Google Colab: `Archivo > Abrir notebook > GitHub` y pega la URL.
4. Ejecuta las celdas en orden con `Shift + Enter`.

### Opción 2 — Clonar el repositorio localmente

```bash
# Clonar el repositorio
git clone https://github.com/andresenvigado-jpg/proyecto-de-prueba.git

# Ingresar al directorio
cd proyecto-de-prueba

# Instalar dependencias (si aplica)
pip install -r requirements.txt
```
